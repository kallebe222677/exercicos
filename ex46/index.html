Codigos de terminal E pra que serve

  • date:exibir data e hora atuais

  • cal:exibir um calendario

  • uptime:exibir há quanto tempo vocẽ não desliga/reinicia seu computador

  • pwd:exibir qual é o diretorio atual

  • cd:muda o diretório atual

  • mkdir:criar uma pasta,e tambem cria pasta e subdiretorios (de uma só vez)

  • ls:exibir arquivos/diretórios do diretorio atual.

  • move:move ou (renomea) um arquivo ou pasta

  • del:deleta um arquivo lembrando Cuidado: o terminal não pede confirmação antes de deletar; tome muito cuidado para não deletar o arquivo errado!

  • rmdir:deletar uma pasta vazia

  • rmdir /s:deletar uma pasta que não esteja vazia

  • type: exibir o conteúdo de um arquivo

  • cls:limpar o treminal
